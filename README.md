#### int取值范围  
|类型    | 取值范围      | 
| ------ | ------        | 
| int8   | -128到127     | 
| uint8  | 0到255        |
| int16  | -32768到32767 |
| uint16 | 0到65535      |
| int32  | -2147483648到2147483647  |
| uint32 | 0到4294967295 |
| int64  | -9223372036854775808到9223372036854775807 |
| uint64 | 0到18446744073709551615 |
 
int和uint是根据 CPU 获取取值范围   
CPU 型号：runtime.GOARCH  
int的长度：strconv.IntSize
